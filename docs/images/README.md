# Capturas de Ejecución

Esta carpeta debe contener las capturas de pantalla que demuestran la ejecución correcta del proyecto.

## Tipos de Capturas Requeridas:

### Compilación y Construcción
- `compilacion_exitosa.png`: Captura mostrando la compilación sin errores
- `construccion_maven.png`: Captura del comando `mvn clean compile`

### Ejecución de Pruebas
- `pruebas_unitarias.png`: Captura de la ejecución de todas las pruebas
- `cobertura_codigo.png`: Captura del reporte de cobertura (si aplica)

### Ejecución de la Aplicación
- `ejecucion_main.png`: Captura de la ejecución del método main
- `salida_consola.png`: Captura de la salida esperada en consola

### IDE y Estructura
- `estructura_proyecto.png`: Captura mostrando la estructura completa del proyecto
- `editor_codigo.png`: Captura del código principal en el IDE

## Formato de Archivos:
- Preferiblemente PNG para mejor calidad
- JPG aceptable para capturas grandes
- Resolución mínima recomendada: 1024x768

## Nomenclatura:
Usar nombres descriptivos en minúsculas con guiones bajos.
